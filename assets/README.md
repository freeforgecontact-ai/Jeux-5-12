# assets/ — Banque d'assets

Les visuels et sons **générés** (Gemini) avant intégration dans les jeux. Voir `docs/04_PIPELINE_ASSETS.md` et `prompts/images/`.

- `characters/` — mascotte Lumi, personnages réutilisables.
- `backgrounds/` — décors (format paysage 16:9).
- `ui/` — icônes, boutons.
- `logos/` — logo couleur, monochrome, icône carrée (192px & 512px).
- `audio/` — effets et musiques douces.

**Nommage** : minuscules, sans accents ni espaces (ex. `personnage_lumi_face.png`).
**Rappel** : générer avec **fond uni** → **détourer** → optimiser (Gemini ne fait pas de vraie transparence).
