# website/ — Le portail (hub)

La page d'accueil publique qui **liste et lance tous les jeux**. C'est ici que les parents arrivent.

À contenir (voir `docs/08_LANCEMENT_MARKETING.md`) :
- Catalogue **filtrable par âge et par matière**.
- **Une page par jeu** (optimisée SEO, FR + EN) : aperçu, âge, compétence, bouton « Jouer ».
- Page **« Pour les parents »** (porte parentale), **politique de confidentialité**, page **« Financement »**.
- Badges de confiance : *Sans pub · Sans pistage · 100 % gratuit · Bilingue*.
- Bouton **« Recommander à un parent »**.

> Hébergé gratuitement sur **Cloudflare Pages** (bande passante illimitée).
